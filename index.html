Products

GET
/products
Retrieve all products

List all Products in your organisation.

Parameters

NameInTypeDescription
organisationId* header string Your organisation Id

Responses

200 OK

Response Body: array of Product-Summary

A summary of a product, suitable for lists.

POST
/products
Create a new product

Creates a new Product. A templateId can be provided to base the new product on an existing template.

Parameters

NameInTypeDescription
organisationId*headerstringYour organisation Id

Request Body

PropertyTypeDescription
name*stringThe name of the new product.
templateIdstringThe ID of a template to use for creation.

Responses

200 OK

Product created successfully.

GET
/products/{productId}
Retrieve a single product

Get detailed information about a single Product by its ID.

Parameters

NameInTypeDescription
productId*pathstringThe unique identifier for the product.
organisationId*headerstringYour organisation Id

Responses

200 OK

Response Body: Product

The full product object.

Parameters

NameInTypeDescription

Responses

200 OK

POST
/linked-organisations
Create a new Linked Organisation

Creates a new Linked Organisation with the given name and contact email.

Parameters

NameInTypeDescription
organisationId*headerstringYour organisation Id

Request Body

PropertyTypeDescription
name*stringThe name of the organisation to link.
organisationContactEmailstringThe primary contact email for the linked organisation.

Responses

200 OK

Linked Organisation created successfully.
